#Templito
======

Temp·li·to [temp-**lee**-toh]
  1 *(noun)* A small underscore.js template precompiler.
  2 *(noun)* A template burrito.

Generates javascript files for underscore.js templates.

To install:

```bash
npm install -g templito
```

For quick help:

```bash
templito
```

will bring up help info.
